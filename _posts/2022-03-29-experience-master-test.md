<font size = "5"><p>先来简单介绍一下答主的个人情况吧。答主本科就读于浙江某垃圾双非，本科期间干啥啥不行旷课第一名，基本上啥课都是飘过(4级也是430+)。本科期间有过acm经历，拿过两块区域赛银，然后大三上的时候去了字节跳动实习。实习了三个月还是觉得读书好啊(打工？这辈子都不会打工的), 最后遂选择考研，最后初试总分381分，英一76，政治67，数一119，专业课119。希望这篇帖子能对你产生帮助。(终于有机会写经验贴装个B了)</p></font>

<font size = "5"><p>择校的话，没啥好讲的，当时非985不上，去掉特别好的几所，去掉特别差的几所，然后几乎是投色子一样投中了川大网安。后来人请勿学习。</p></font>

<font size = "5"><p>然后讲一下英语吧，英语反正我就是从3月份开始在红宝书上背英语单词，按照那啥艾宾浩斯曲线啥的，每天保持新背1-2个unit。具体多少可以视情况调整。当时基本上是从7：30背到10：00左右。背第二遍就会快一点，早上多出来的时间就开始看看田静的语法课，这个老师讲的还是不错的，尽管最后看完发现不认识的句子还是不认识，好好背单词才是王道。然后到大概暑假刚开始左右的时间，我开始对历年真题阅读部分进行精翻，就是逐句翻译，然后对照黄皮书上的翻译进行修改。个人认为这一步还是挺有用的，一方面锻炼了读句子的能力便于做阅读，另一方面还练习了翻译的能力。做完阅读精翻，之后就二刷阅读，这次就是要掐表模拟考场来做了。阅读的老师嘛，当时看的是唐迟，觉得黄皮书答案解释的不到位的时候可以看他的视频解析。emmm，完型的话我觉得没必要看课来练习了，这个提升真不大，印象中10以前的完型难的一比，后面的只要你单词看得懂就能做。翻译跟的唐静，这老师真的是强推啊，认真负责，讲课也舒服。最后新题型也很重要，你们可以多看看那些技巧，但是我一直到考前新题型都做不明白，所以我也就不说了。其实英语最后总结下来真的只有背背背，单词过个6、7遍，基本上你就是神了</p></font>

<font size = "5"><p>政治的话，全程跟着肖秀荣即可。政治可以晚一点再开，比如9、10月份都来得及。然后视频课的话我个人是只看了马原的徐涛视频，剩下的中特、思修之类的完全觉得可以在1000题上边学边练，有条件可以去淘宝上买一本那种什么政治选择一本通，那些会整理好政治1000对应的知识点，对于做1000还是挺方便的。这个时候可能有人会问腿姐，毕竟知乎上腿姐吹的是铺天盖地。我的看法是，可以但没必要，个人认为这个老师还是过度营销了，他的那些技巧都是再平常不过的文科题目做法，高中学过政治都会的。课程还长的狠，政治我觉得只要在肖四出来前把1000题刷个2-3遍足以了。之后就是好好看肖4肖8，那些其他老师的模拟卷我也不是很推荐做，除非你最后时间很多，因为那些卷子的质量说实话真的不如肖4肖8，有些题目你真的觉得出的有些无理取闹，这种题目你错了你能恶心好久。最后的最后，那就是至关重要的肖4大题了，那段时间一定要花时间去背，数学英语专业课时间都可以减下来，把肖四答题背熟，基本上就够了。至于你可能会说，今年肖老压题就没中，其实我想说的是肖4的背诵基本上已经给你了写的素材加上结合材料。到时候即便出题人反压题，基本上你还是都写得出来的，除了今年可能就那个马原的题目没有完全压中，但是你不会别人也不会，不用太在意。最后，考研政治的空卡，跟他就是交纯纯的智商税，避开，自己政治都没学明白呢，还教别人呢。</p></font>

<font size = "5"><p>嘿嘿，终于开讲数学了。先说一下我的准备过程，再说感悟把。我是3月份看的汤家凤的高数基础课视频，然后写的1800高数的基础。之后跟的李永乐的线代，写的660的线代部分(不过660的线代好像太简单了，大家还是别做了)，所以后来我又做了1800里面的基础线代。在之后跟了余炳森的概率，写的也是他讲义上的题目。基础部分就做了这些。然后我强化的话，没有看老师视频，觉得有点浪费时间了，所以买了本李正元的复习全书，这本书我吹爆好吧，写的是真的好。第一遍过全书写的上面的例题，写完一章做一章李林880的对应题目。然后第二遍过全书，再做一遍上面的例题，和之前880上的错题。线代强化自然还是跟着永乐大帝的视频，写着永乐的讲义和880的线代强化。概率最后强化我是看了一下方浩的视频，写的也还是880。至于最后就是疯狂刷模拟卷，算一算大概有将近40涨了吧。列一下我写过的：李正元的5套卷、张宇8+4、共创超越10套、李艳芳3套、余炳森5套、李林6+4。

然后讲一下，上面这些过程的感悟吧。首先需要明确的一点，数学是一定要投入大量时间进去的。我刚开始备考的时候，基本上一台能除了背英语单词，剩下的时间全部都在写数学。基础第一遍看视频的时候，强烈建议边看边记一下笔记。应为数一内容很多，你学完线代，高数可能就忘了，自己的笔记不仅有助于建立起一个知识框架，对快速回忆之前学过的知识点也有作用。然后第一遍遇到想了很久都做不出来的题目，不要死磕，先跳过，有的时候学完强化再回来看的时候就豁然开朗。至于到了强化，就基本不需要再记笔记了，毕竟大部分理论知识你都已经会了，学的是一些更高级的解题技法而已。这种技法，靠记是记不牢的，唯有多做多练，自然能融会贯通。还有当你强化，结束的时候，我推荐这个时候拿几张8k的纸，对着全书过一遍知识点，把一些冷门的，你不熟的、和一些需要硬背的知识点以及解题的一些常用思路都记在上面，之后做模拟卷的时候也补充到这堆纸里，最后复习的时候看这些纸就行了。然后说说模拟卷的选择吧，我这里强推余炳森5套、合工大共创超越。余炳森的小题出的真的是好，共创超越我觉得大题出的比较好。然后是李林的6+4，这个中规中矩，难度也适中，应该适合绝大部分人。至于22年张宇的4+8高分版，个人觉得出的很水没啥必要。李正元的重考察概念，大题的题型和现在的题型出入还是有点大的。然后李艳芳的卷子过于难了，大家别给自己找不自在了。

然后再说说一向争议很大的考研数学老师选择的问题。如果你不想花时间去选老师，跟武忠祥就行，至少身边同学反映都还不错，我个人只看过其强化，讲的还是可以的。汤老师也很好，但是他有些课会放在直播里讲，这就很难受。而且据说强化一般。我想说的是，考研数学最终还是多做多练，别一学不会就怪老师。
</p></font>

<font size = "5"><p>专业课来啦。今年第一年考872，内容是os，计网、安全。这个说到川大专业课辅导，我先来讲一下基本上考川大的都知道的机构川流不息。这个机构打广告是真的还挺猛的，所以我当时也是直接买了资料。对于资料，我的评价是质量不太行。首先答案是漏洞百出，做真题对答案跟扫雷一样。但是毕竟有资料总比没有好，至少题目是没问题的。然后就是他们机构出的模拟题，这个我觉得的就是你有钱有时间买来可以玩玩，但是没有必要。毕竟是一群学生网上找的题目改一下，我觉得工科的题目学生自己都没全学明白，还想出题，简直是天方夜谭。不过他们提供的找学长学界的服务，还是有用的，我当时找到一位很热心的学姐，对我帮助还是挺大的。当然，如果你有门道联系到往年学长学姐愿意给你答疑的你就不用找机构了。总体评价是有一定辅助作用，但是价格对于普通学生而言还是偏贵了，远不如他们宣传的那么好。

872的话，整体难度是不如408的，重点在计网，考察的比较细，比408的难，然后要注重教材，单看王道不太够的。然后说说我的备考吧，因为当时7月之前还只是考一门计网，所以开始我就只是看谢希仁的那本计网，然后边看边做王道和谢的课后题。之后就是再看自顶向下那本黑书上的课后题，这本书比较难，有些课后题实在看不懂就过吧。基础的题目一定要做一做，872可能会在里面出原题的。然后操作系统的话，我是暑假开始看的，就用王道的书够了。但看今年os的难度是远不如408的，王道刷个两三遍就没问题了。安全的话，因为安全就30分，内容却很庞杂，当时也没啥资料，所以几乎我就没怎么看安全。就最后看一看谢希仁书上第七章和自己搜了一点安全的资料准备了一下。然后有时间你们可以关注一下时下热门的话题，什么大数据、云计算、区块链等等的话题，因为往年都会出一道关于这些内容的论述题，稍微准备一下，别到时候遇到了一点话写不出。不过也不要太花时间到这题上，我理解的是这种题都是老师用来控分用的。最后说一下，冲刺阶段的专业课该如何学习，我当时开始做那个往年真题，感觉风格非常的诡异，很多题目我完全不会。后来我就去找他们本科生的期末卷做，这个就舒服很多，也和今年卷子的风格比较像。反正去年几乎除了一道原题在他们的期末卷里面。872之前的考试内容也是改过好几次，往年的真题参考价值有限，所以多多注重期末卷和参考书。至于os，看看计院874的os部分就行了。最后给个我考研自己整理的一些笔记和资料在这把。</p></font>
